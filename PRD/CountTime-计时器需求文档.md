# CountTime 计时器 - 需求文档

## 1. 项目概述

基于 Python PyQt5 的桌面计时器应用，提供简洁美观的 GUI 界面，支持开始、暂停、重置等基本计时功能。

## 2. 技术栈

- **语言**: Python 3.13
- **GUI 框架**: PyQt5 5.15.11
- **启动方式**: start.bat 批处理脚本

## 3. 功能需求

### 3.1 核心功能
| 功能 | 说明 |
|------|------|
| 开始计时 | 点击"开始"按钮，从 00:00:00 开始每秒递增 |
| 暂停计时 | 点击"暂停"按钮，暂停当前计时 |
| 重置计时 | 点击"重置"按钮，归零并停止计时 |

### 3.2 界面要求
- 深色主题（#1a1a2e 背景）
- 大字体时间显示（72px 等宽字体，绿色 #00ff88）
- 状态提示标签（显示当前状态：计时中/已暂停/已重置）
- 三个操作按钮（开始-绿色、暂停-橙色、重置-红色）
- 固定窗口大小 500x300

### 3.3 交互逻辑
- 开始后"开始"按钮禁用，"暂停"按钮启用
- 暂停后"暂停"按钮禁用，"开始"按钮启用
- 重置后恢复初始状态
- 状态标签实时反映当前操作状态

## 5. 启动方式

双击 `start.bat` 即可启动，脚本会自动：
1. 检查 Python 环境
2. 检查/安装 PyQt5 依赖
3. 启动计时器窗口
